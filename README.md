# 🎭 Face Mesh Project

A real-time **Face Mesh Detection** application built using **Python**, **OpenCV**, and **MediaPipe**. This project detects facial landmarks and draws a detailed mesh over the face using a webcam feed.

## ✨ Features

* 🎯 Real-time face detection
* 😊 468 facial landmark tracking
* 🕸️ Face mesh visualization
* 📷 Webcam-based processing
* ⚡ Fast and lightweight performance

## 🛠️ Technologies Used

* 🐍 Python
* 👁️ OpenCV
* 🤖 MediaPipe

## 🚀 Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/mishikabagrecha/face-mesh-project.git
cd face-mesh-project
```

### 2️⃣ Create a Virtual Environment

```bash
python -m venv venv
```

### 3️⃣ Activate the Virtual Environment

**💻 Mac/Linux**

```bash
source venv/bin/activate
```

**🪟 Windows**

```bash
venv\Scripts\activate
```

### 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run the application:

```bash
python face.py
```

📸 Your webcam will open and display facial landmarks in real time.

## ⚙️ How It Works

* 📷 Captures live video from your webcam using OpenCV.
* 🧠 Uses MediaPipe Face Mesh to detect facial landmarks.
* 🎨 Draws a detailed mesh over the detected face.
* ⚡ Updates the mesh in real time for smooth tracking.
